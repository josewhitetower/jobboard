# jobboard

A simple Job Board with user authentication

## Requirements
1. [Node.js and npm](http://nodejs.org)
2. [MongoDB](http:///mongodb.org)



## Install

1. Clone this repo
```bash
git clone https://github.com/josewhitetower/jobboard.git
```

2. Run the client side:
```bash
cd client
npm run dev
```

3. Run the server side
```bash
cd server
npm start
```
